# ☕ CODSOFT - Java Programming Internship

Welcome to my **Java Programming Internship** repository completed under **CodSoft**.

This repository contains the Java projects I developed during my internship. Each project focuses on strengthening core Java concepts, problem-solving skills, and Object-Oriented Programming (OOP) through hands-on implementation.

---

## 📌 Projects Included

### 🎯 Task 1 - Number Game
A console-based number guessing game where the user has to guess a randomly generated number within a limited number of attempts.

**Features**
- Random number generation
- Limited attempts
- High/Low hints
- Score tracking
- Play multiple rounds
- Input validation

**Concepts Used**
- Loops
- Conditional Statements
- Random Class
- Scanner Class

---

### 📊 Task 2 - Student Grade Calculator
A Java application that calculates total marks, average percentage, and assigns grades based on student performance.

**Features**
- Multiple subject marks input
- Total marks calculation
- Percentage calculation
- Grade generation
- Pass/Fail status
- Input validation

**Concepts Used**
- Loops
- Conditional Statements
- Scanner Class

---

### 🏧 Task 3 - ATM Interface
A console-based ATM simulation that performs basic banking operations using Object-Oriented Programming.

**Features**
- Balance Inquiry
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-driven Interface

**Concepts Used**
- OOP
- Classes & Objects
- Methods
- Scanner Class

---

### 💱 Task 4 - Currency Converter
A real-time currency converter built using Java HTTP Client API.

**Features**
- Live Exchange Rates

- Multiple Currency Conversion
- Exception Handling
- Console-based Interface

**Concepts Used**
- Java HTTP Client

- Exception Handling
- Scanner Class
